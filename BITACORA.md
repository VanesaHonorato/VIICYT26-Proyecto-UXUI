# Bitácora del Proyecto

## Semana 1
**Periodo:** 15 - 19 de junio

### Objetivo
Configurar el entorno de trabajo y preparar las herramientas necesarias para el desarrollo del proyecto.

---

## Actividades realizadas

### Configuración del repositorio
- Se creó el repositorio del proyecto en GitHub.

### Configuración del entorno
- Instalación y configuración de Git.

- Configuración de Claude Desktop.

- Se configuró el servidor MCP.

---

## Observaciones

Durante esta semana se dejó preparado el entorno de desarrollo para comenzar con el diseño de la interfaz en las siguientes etapas del proyecto.


---


## Semana 2

**Periodo:** 22 - 26 de junio

## Objetivo

Investigar el proceso de diseño UX para definir la estructura de navegación de la plataforma y establecer las bases para el desarrollo de la interfaz de usuario.

## Actividades realizadas

- Se investigaron los conceptos básicos de User Experience (UX).
- Se utilizó Claude como herramienta de apoyo para la generación de propuestas de diseño.
- Se redactó un prompt para generar la estructura del Sitemap considerando los diferentes tipos de usuarios y las funcionalidades del sistema.
- Se obtuvo un Sitemap inicial, el cual fue revisado y adaptado por el equipo.
- A partir del Sitemap se desarrolló el User Flow de la plataforma.
- Se realizaron wireframes de baja fidelidad en el pizarrón para visualizar la distribución de las principales pantallas y módulos del sistema.
- Se discutió en equipo la arquitectura de navegación y la organización de las pantallas.

## Evidencias

- Sitemap.
- User Flow.
- Wireframes de baja fidelidad.
- Prompt utilizado en Claude.

## Observaciones

Durante esta semana se definió la estructura de navegación de la plataforma y se establecieron las bases para el diseño de la interfaz. El uso de Claude permitió agilizar la generación de propuestas que posteriormente fueron revisadas y ajustadas por el equipo.
