## Prompt 1 - Generación del Sitemap

### Objetivo

Generar la estructura de navegación del sistema y definir las pantallas necesarias para cada tipo de usuario.

### Prompt utilizado

Genera el sitemap con estas especificaciones. Requiero que generes todas las pantallas que serán necesarias para representar el flujo de usuarios y las funcionalidades que se muestran, de manera que me generes mi sitemap con ejemplos y comentarios que me faciliten el entendimiento de cada pantalla o sección. Si es posible toma en cuenta que se pueden repetir pantallas para diferentes usuarios.

### Resultado

Claude generó una propuesta inicial del Sitemap del sistema, la cual fue utilizada como base para elaborar el flujo de navegación y posteriormente fue revisada y adaptada por el equipo.

## Observaciones

El uso de inteligencia artificial permitió acelerar la generación de una primera propuesta de arquitectura de información. Posteriormente el equipo realizó ajustes para adaptarla a las necesidades específicas del proyecto.
